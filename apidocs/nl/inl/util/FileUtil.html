      • detectBomEncoding

        public static String detectBomEncoding(BufferedInputStream inputStream,
                       String defaultEncoding)
                                throws IOException
        Detect the Unicode encoding of an input stream by looking for a BOM at the current position. If no BOM is found, the specified default encoding is returned and the position of the stream is unchanged. If a BOM is found, it is interpreted and the corresponding encoding is returned. The stream will remain positioned after the BOM. This method uses InputStream.mark(), which must be supported by the given stream (BufferedInputStream supports this). Only works for UTF-8 and UTF16 (LE/BE) for now.
        Parameters:
        inputStream - the input stream
        defaultEncoding - encoding to return if no BOM found
        Returns:
        the encoding
        Throws:
        IOException
