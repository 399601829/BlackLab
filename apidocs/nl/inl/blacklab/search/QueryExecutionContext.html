nl.inl.blacklab.search

Class QueryExecutionContext

  • java.lang.Object
    • nl.inl.blacklab.search.QueryExecutionContext

  • public class QueryExecutionContext
extends Object
    Represents the current "execution context" for executing a TextPattern query. Inside a query, this context may change: a different property may be "selected" to search in, the case sensitivity setting may change, etc. This object is passed to the translation methods and keeps track of this context.
