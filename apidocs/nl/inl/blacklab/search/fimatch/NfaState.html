Class NfaState

  • java.lang.Object
    • nl.inl.blacklab.search.fimatch.NfaState
  • Direct Known Subclasses:
    NfaStateAnd, NfaStateAndAcyclic, NfaStateMatch, NfaStateMultiTermPattern, NfaStateNot, NfaStateOr, NfaStateOrAcyclic, NfaStateToken

    public abstract class NfaState
extends Object
    Represents both a state in an NFA, and a complete NFA with this as the starting state. (Note that the "matching state" is simply represented as null in the NFAs we build, which is convenient when possibly appending other NFAs to it later)
