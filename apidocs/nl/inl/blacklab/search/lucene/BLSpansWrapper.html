      • blSpanQueryFrom

        @Deprecated
public static BLSpanQuery blSpanQueryFrom(org.apache.lucene.search.spans.SpanQuery spanQuery)
        Deprecated. moved to BLSpanQuery and renamed to wrap().
        Rewrite a SpanQuery after rewrite() to a BLSpanQuery equivalent. This is used for BLSpanOrQuery and BLSpanMultiTermQueryWrapper: we let Lucene rewrite these for us, but the result needs to be BL-ified so we know we'll get BLSpans (which contain extra methods for optimization).
        Parameters:
        spanQuery - the SpanQuery to BL-ify (if it isn't a BLSpanQuery already)
        Returns:
        resulting BLSpanQuery, or the input query if it was one already
