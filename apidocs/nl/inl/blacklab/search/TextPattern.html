    public abstract class TextPattern
extends Object
implements Cloneable
    Describes some pattern of words in a content field. The point of this interface is to provide an abstract layer to describe the pattern we're interested in, which can then be translated into, for example, a SpanQuery object or a String, depending on our needs.
